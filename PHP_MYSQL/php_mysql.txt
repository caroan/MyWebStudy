주요 함수

mysqli_connect() : MySQL 서버에 연결한다.
mysqli_connect_error() : MySQL 서버에 연결 오류 발생 시 그 원인을 알려준다.
mysqli_close() : MySQL 서버에 연결된 것을 종료한다.
mysqli_select_db() : 사용할 데이터베이스를 지정한다.
mysqli_query() : SQL 문을 서버에 실행한다.
mysqli_error() : SQL 문이 서버에서 실패한 경우 그 원인을 알려준다.
mysqli_num_rows() : SELECT문의 결과가 몇개의 행인지 알려준다.
mysqli_fetch_array() : SELECT문의 실행 결과에서 결과 행을 추출한다.
